## studievragen 
### 3 dingen die je hebt geleerd
1. Vandaag was ook de CSS workshop met als thema grid. Ik heb heel veel over grid geleerd, zoals: 
* Je kunt een responsive grid maken met `auto-fit`
* Alleen de directe kinderen van een element 'reageren' op het grid
* Je kunt zowel `grid-column-start` als `grid-area` gebruiken

2. Daarnaast heb ik onderzoek gedaan naar hoe een zoek functie toe te passen in de website, hiervoor heb ik een tutorial gevonden en deze nagemaakt, zie hier: https://julia-stevens.github.io/i-love-web/

3. Grid kun je zowel toepassen op macro- als micro lay-out
* Zo heb ik een grid toegepast om de cards die ik op de squad page wil gaan gebruiken: https://github.com/julia-stevens/your-tribe-squad-page/blob/f193828c2e38df2556158df9af1eeeccd6478c82/styles/julia-styles.css#L166-L321

### 2 dingen die je interessant vindt
1. Ik vind het interessant om meer over grid te leren, omdat het het positioneren van elementen op de pagina zoveel makkelijker maakt. Daarnaast vind het heel interessant om een responsive grid toe te passen en hiermee te experimenteren 
2. Ik vind het ook interessant om bezig te zijn met de HTML elementen, hiervoor heb ik tijdens het maken van de breakdown schets de MDN pagina opgehouden en gezocht naar, wat ik denk dat juiste elementen zijn.
